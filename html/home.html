Memcache

Qué es memcache

Memcache es un motor de bases de datos en memoria basado en pares llave-valor (como Redis).

Memcache se ofrece como un servicio mediante el API del AppEngine de Google.

Por y para qué

Rendimiento y costos

  • Tokens
  • Llamados a APIs (como este app)
  • Contadores
  • Configuraciones
  • Lecturas con menor latencia...

Cómo usarlo

En Java, utilizando el API del AppEngine o de JCache

En Go y Python utilizando el modulo de memcache del AppEngine
